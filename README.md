# project4
# Array Operations Menu Driven Program in Python

## Description

This project is a menu-driven Python program that performs different operations on arrays using functions.

The program demonstrates:

* Built-in functions
* User-defined functions
* Lambda functions
* Recursion
* *args and **kwargs
* Global variables
* 1D and 2D list operations

---

# Features

1. Display length, sum, and maximum value of array
2. Calculate average of elements
3. Find duplicate values
4. Find factorial using recursion
5. Double values using lambda function
6. Sort array values
7. Display 1D list
8. Demonstrate *args
9. Demonstrate **kwargs
10. Exit program

---

# Technologies Used

* Python 3

# Functions Used

## math(arr)

Displays:

* Length of array
* Sum of elements
* Maximum value

## avgg(arr)

Calculates average of array elements.

## duplicate(arr)

Finds duplicate values in array.

## unique_val(arr)

Displays unique values.

## factorial(n)

Calculates factorial using recursion.

## double(arr)

Doubles array values using lambda function.

## count()

Uses global variable counter.

## val_args(*args)

Demonstrates use of *args.

## val_kwargs(**kwargs)

Demonstrates use of **kwargs.

## sort_val(arr)

Sorts array values.

## data(arr)

Displays 1D list.

## data_1(arr)

Displays 2D list.
